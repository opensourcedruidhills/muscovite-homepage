Detailed Description

Accumulating validation error collection for the three-tier architecture.

ValidationErrors collects field-level errors across all three validation tiers without short-circuiting. Each error carries the field name, a human-readable message, the tier that produced it, and an optional machine-readable error code.

Usage:

ValidationErrors errors;
errors.add("age", "must be positive", ValidationTier::strong_type);
errors.add("birth_date", "must not be in the future", ValidationTier::entity_validator);
if (errors.has_errors()) {
auto json = errors.to_json(); // structured JSON for gRPC details
}
