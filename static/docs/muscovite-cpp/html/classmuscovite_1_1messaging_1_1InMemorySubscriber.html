Detailed Description

In-memory subscriber for unit testing

Test code calls deliver() to push messages to registered handlers. Thread-safe.

Usage:

InMemorySubscriber sub;
sub.connect();
auto subscription = sub.subscribe("muscovite.patient.>", [](const auto& env) {
// handle message
return HandlerResult::Ack;
});
sub.deliver(envelope); // invokes matching handlers
CHECK(sub.messages_received() == 1);
