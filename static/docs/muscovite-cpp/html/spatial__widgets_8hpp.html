Detailed Description

Qt coordinate input widgets for spatial queries.

Provides specialized input widgets for geographic coordinates:

  • LatLonInput: Latitude/longitude pair with range validation
  • RadiusInput: Distance in kilometers with range validation
  • BoundsInput: Min/max latitude/longitude for bounding box queries
  • CoordinateValidator: Custom validator for coordinate format and range

All widgets integrate with the Qt validation framework (validation_codes.hpp).
