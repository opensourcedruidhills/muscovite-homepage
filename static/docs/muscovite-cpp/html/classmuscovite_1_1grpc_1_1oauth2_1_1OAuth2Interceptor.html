Detailed Description

OAuth2 Interceptor for gRPC channels.

Implements ::grpc::MetadataCredentialsPlugin to automatically add Authorization header with bearer token to every gRPC call.

Supports both built-in authentication (muscoviteserver security module) and external OAuth2 providers (Keycloak, Auth0, Azure AD).

Features:

  • Automatic token refresh before expiry
  • Dual-mode authentication (built-in + OAuth2)
  • Secure token storage (memory for access, keychain for refresh)
  • Token revocation on logout
