Detailed Description

spdlog custom flag formatter reading the thread-local correlation ID.

Register with pattern_formatter::add_flag<CorrelationFlagFormatter>('') then use `% in your spdlog pattern string. Outputs the current CorrelationContext correlation ID, or -` when no request is active.

Thread-local access is lock-free — zero overhead on the hot path.

auto formatter = std::make_unique<spdlog::pattern_formatter>();
formatter->add_flag<muscovite::grpc::CorrelationFlagFormatter>('*');
formatter->set_pattern("[%Y-%m-%d %H:%M:%S.%e] [%l] [%n] [cid=%*] %v");
spdlog::set_formatter(std::move(formatter));
