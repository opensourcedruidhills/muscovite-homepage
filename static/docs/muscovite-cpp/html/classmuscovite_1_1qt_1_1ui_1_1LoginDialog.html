Detailed Description

Login dialog for authenticating users.

Provides a simple username/password login UI that integrates with AuthManager. Supports:

  • Username/password input
  • Remember me functionality
  • Login progress feedback
  • Error messages

Usage:

LoginDialog dialog;
dialog.setServerAddress("localhost:50051");
if (dialog.exec() == QDialog::Accepted) {
// User is authenticated
auto session = dialog.session();
}
