Detailed Description

RAII wrapper for cnats natsConnection handle.

See also
FTR-208 — NATS RAII connection wrappers
CPP-PATTERNS.md §17 — RAII for all resources

Wraps the raw natsConnection* C handle in a std::unique_ptr with a custom deleter that calls natsConnection_Destroy() on scope exit. Move-only (deleted copy). Use release() when the C API needs to take ownership (e.g., callback registrations that outlive the wrapper).
