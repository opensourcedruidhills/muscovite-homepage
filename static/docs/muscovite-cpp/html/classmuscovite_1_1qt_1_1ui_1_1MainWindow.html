Detailed Description

Main application window after successful authentication.

Displays:

  • Authenticated user information (username, groups, privileges)
  • Tabs for different modules (initially empty, to be populated by services)
  • User menu with logout option
  • Refresh token indicator

Usage:

auto window = std::make_unique<MainWindow>();
window->setAuthManager(&AuthManager::instance());
window->show();
