Detailed Description

template<typename T>
class muscovite::qt::models::GrpcItemModel< T >

Base class for gRPC-backed Qt item models.

Base class for gRPC-backed item models.

Provides:

  • QAbstractTableModel implementation for Qt Widgets (QTableView) and QML
  • gRPC channel and auth manager integration
  • Loading/error state management with signals
  • Column headers and Qt role names derived from DSL fields

Subclasses (generated per-entity) must implement:

  • fetch() to call gRPC List method and populate rows
  • Optionally override headerData() for custom column titles

Usage with QSortFilterProxyModel: auto* model = new ChemicalElementModel(this); auto* proxy = new QSortFilterProxyModel(this); proxy->setSourceModel(model); proxy->setSortRole(Qt::UserRole); // Sort by data, not display tableView->setModel(proxy);

Provides common functionality for QAbstractTableModel subclasses that display data from gRPC services.
