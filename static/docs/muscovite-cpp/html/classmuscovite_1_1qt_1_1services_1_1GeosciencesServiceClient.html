Detailed Description

Example Geosciences service client.

Demonstrates how to create a service client that:

  • Inherits from ServiceClientBase for auth-aware gRPC calls
  • Makes unary calls with automatic authentication headers
  • Handles errors and retries
  • Emits signals for async results
