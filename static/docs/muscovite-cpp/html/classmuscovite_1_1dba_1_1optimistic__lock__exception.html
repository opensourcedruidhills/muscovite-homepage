Detailed Description

Thrown when an UPDATE fails due to a stale version (optimistic locking).

The entity was modified by another transaction between read and write. The caller should reload the entity and retry or present a conflict dialog.

Generated by CommandRepository::update() when: UPDATE ... SET version = version + 1 WHERE id = $1 AND version = $2 returns zero rows.
