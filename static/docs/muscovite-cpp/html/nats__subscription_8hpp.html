Detailed Description

RAII wrapper for cnats natsSubscription handle.

See also
FTR-208 — NATS RAII connection wrappers
CPP-PATTERNS.md §17 — RAII for all resources

Wraps the raw natsSubscription* C handle in a std::unique_ptr with a custom deleter that calls natsSubscription_Destroy() on scope exit. Move-only (deleted copy).
