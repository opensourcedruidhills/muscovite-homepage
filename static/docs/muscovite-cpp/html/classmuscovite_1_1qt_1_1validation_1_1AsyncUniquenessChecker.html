Detailed Description

Interface for async uniqueness checking.

Implementations wrap gRPC calls to check if a field value already exists. The check excludes the current entity (when editing) to allow saving without changing the unique field.

Example usage in generated validator:

auto checker = std::make_unique<PatientMrnUniquenessChecker>(grpcChannel);
FieldValidationRule::asyncUnique(
mrnEdit_,
tr("MRN"),
[checker = checker.get(), currentId = entityId_](
const QString& value,
std::function<void(bool, const QString&)> callback
) {
checker->checkUniqueness(value, currentId, callback);
}
);
