Detailed Description

Bounding box input widget for bounded spatial queries.

Features:

  • Four QDoubleSpinBox widgets (min/max latitude and longitude)
  • Automatic validation that min <= max
  • Clear labeling of each corner
  • Can generate ST_MakeEnvelope parameters for PostgreSQL

Layout: [ Min Latitude | spinbox ] [ Max Latitude | spinbox ] [ Min Longitude | spinbox ] [ Max Longitude | spinbox ]
