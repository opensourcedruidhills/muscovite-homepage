Detailed Description

Transport-agnostic search indexer for processing domain events.

The SearchIndexer receives domain events (via process_event()) and delegates to registered DocumentMappers for entity-specific logic. It is intentionally decoupled from NATS — the transport layer is wired by generated code that converts EventContext → SearchEvent.

Architecture:

NatsEventConsumer (dba) ← transport layer (generated wiring)
│
▼ SearchEvent
SearchIndexer (search) ← this class
│
├── DocumentMapper::fetch_and_map() → OpenSearchClient::index_document()
└── (Deleted events) → OpenSearchClient::delete_document()
