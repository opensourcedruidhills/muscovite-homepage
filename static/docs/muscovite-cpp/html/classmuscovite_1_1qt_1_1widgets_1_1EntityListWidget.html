Detailed Description

Base class for entity list widgets.

Provides a complete list view with:

  • QTableView bound to a GrpcItemModel via QSortFilterProxyModel
  • Search toolbar with text filter and column selector
  • CRUD toolbar with Create/Edit/Delete/Refresh buttons
  • Sorting by clicking column headers
  • Selection handling with signals

Subclasses (generated per-entity) must:

  • Create and set the entity-specific model via setModel()
  • Override createEntity(), editEntity(), deleteEntity() for CRUD actions
  • Optionally customize column visibility and widths

Example usage: auto* listWidget = new ChemicalElementListWidget(channel, authMgr, this); connect(listWidget, &EntityListWidget::entitySelected, this, &MainWindow::onEntitySelected);

Provides a standard layout with QTableView and Add/Edit/Delete buttons.
