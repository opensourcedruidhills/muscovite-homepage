Detailed Description

Thrown when pg_advisory_xact_lock times out.

Another transaction holds the advisory lock on this aggregate and did not release it within the configured lock_timeout.

Generated by CommandService methods when advisory locking is enabled and SET LOCAL lock_timeout is configured.
