Detailed Description

PKCE (Proof Key for Code Exchange) implementation for OAuth2.

RFC 7636 - OAuth 2.0 Proof Key for Public Clients

Used to protect authorization code exchange in public clients (native apps, SPAs) by preventing authorization code interception attacks.
