Detailed Description

Helper for async uniqueness validation via gRPC.

Provides a reusable pattern for checking field uniqueness asynchronously against a gRPC QueryService. Generated validators use this to verify @unique fields don't conflict with existing records.

Per ROADMAP: Qt async uniqueness validation (deferred from v0.8.0)
