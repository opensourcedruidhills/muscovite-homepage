Detailed Description

Compile-time bounded integer template with half-open range support.

BoundedInteger<Min, Max> enforces [Min, Max] range at construction time. Uses std::numeric_limits sentinels for half-open ranges (unbounded side). Returns std::expected<T, std::string> from try_from() factory.
