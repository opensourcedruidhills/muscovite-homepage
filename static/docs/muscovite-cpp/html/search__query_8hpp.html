Detailed Description

Fluent builder for OpenSearch query DSL.

See also
ADR 20260222: OpenSearch Full-Text Search Capability
FTR-116: Search query builder and result types

Provides a type-safe, fluent interface for building OpenSearch JSON queries. Generated code uses this builder instead of constructing raw JSON, ensuring valid query structure and consistent patterns across all contexts.

auto query = SearchQuery::full_text("John Doe")
.filter_term("context", "patient")
.filter_range("birth_date", {.gte = "1990-01-01"})
.paginate(1, 20)
.highlight({"given_name", "family_name"})
.sort("_score", SortOrder::Desc);
auto json = query.to_json(); // valid OpenSearch query JSON
