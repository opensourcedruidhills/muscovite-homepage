Detailed Description

Validation error code registry for Qt dialogs.

Error codes are organized by category:

  • VAL001-009: Required field errors
  • VAL010-049: Format errors (Email, Phone, Postal, etc.)
  • VAL050-079: Range errors (min/max)
  • VAL080-089: Uniqueness errors
  • VAL090-099: Arc constraint errors
  • VAL100+: Custom application errors

All error messages are i18n-ready via tr().
