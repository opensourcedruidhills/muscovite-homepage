Detailed Description

template<typename Entity, typename IdType = typename Entity::pk_type>
class muscovite::dba::Repository< Entity, IdType >

Base repository interface for CRUD operations.

This serves as a base class for generated repositories. Each entity type gets a repository with type-safe operations backed by PL/pgSQL stored procedures.

Template Parameters
EntityThe entity type this repository manages
IdTypeThe entity's identifier type (default: Entity::pk_type)
