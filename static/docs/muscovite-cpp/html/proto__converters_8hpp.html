Detailed Description

Shared protobuf ↔ C++ domain type converter utilities.

See also
ADR 20260303 — C++23 style guide
FTR-207 — Deduplicate uuid_to_string and proto converter utilities

Header-only utilities for converting between protobuf wire format and C++ domain types. Consolidates converters that were previously duplicated across generated gRPC service files.

Primary converters:

  • uuid_to_string / string_to_uuid — UUID ↔ string
  • Timestamp converters (ISO 8601 ↔ system_clock::time_point)
  • Date converters (ISO 8601 ↔ year_month_day)

For higher-level protobuf message ↔ C++ conversions, see type_mapping.hpp. For string-based type converters (Decimal, TimeOfDay), see proto_type_converters.hpp.
