Detailed Description

QML-friendly helper for privilege checking.

This class provides a convenient interface for checking user privileges from QML, enabling conditional UI rendering based on permissions.

Usage in QML:

import Muscovite.Qt 1.0
Button {
text: "Delete"
visible: PrivilegeChecker.has("minerals:delete")
enabled: PrivilegeChecker.has("minerals:delete")
onClicked: mineralService.delete(mineralId)
}
// Or with any of several privileges
Button {
text: "Edit"
visible: PrivilegeChecker.hasAny(["minerals:write", "admin:*"])
}
