Detailed Description

C++20 concepts for DBA entity type constraints (FTR-204).

Provides the DbEntity concept that all generated entity structs satisfy. Use requires DbEntity<T> to constrain templates operating on entities, producing clear constraint-violation errors instead of template walls.
