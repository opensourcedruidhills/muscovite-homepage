Detailed Description

JWT (JSON Web Token) utilities for OAuth2.

Provides JWT parsing and validation for OpenID Connect (OIDC) tokens and custom claims validation.

Note: This is for parsing/validation only. Token signature verification should be done by the authorization server via introspection endpoint or by maintaining a cached JWKS (JSON Web Key Set).
