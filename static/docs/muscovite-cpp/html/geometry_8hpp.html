Detailed Description

Spatial geometry types and operations for PostGIS integration.

This module provides C++ bindings to PostGIS GEOGRAPHY types stored in PostgreSQL. Geometries are transmitted as WKT (Well-Known Text) strings over the network and converted to C++ types as needed.

Architecture: DSL POINT/LINESTRING/POLYGON map to PostGIS GEOGRAPHY(type, 4326) with WGS84 coordinates. All distances are in meters.
