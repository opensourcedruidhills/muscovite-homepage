Detailed Description

Background publisher that polls the outbox and publishes events

Usage:

auto nats = std::make_unique<NatsMessageBroker>(nats_url);
OutboxPublisher publisher(pool, std::move(nats), config);
publisher.start(); // Begins background polling
// ... application runs ...
publisher.stop(); // Graceful shutdown
