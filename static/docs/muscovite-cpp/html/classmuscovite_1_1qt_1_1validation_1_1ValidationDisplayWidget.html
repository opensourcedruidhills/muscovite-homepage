Detailed Description

Widget for displaying validation messages.

Features:

  • QTextBrowser for copyable text (enables bug reporting)
  • Errors shown in red, warnings in orange
  • Messages grouped by field
  • Collapsible design (shows only when has content)
  • i18n-ready via tr()

Usage:

auto* display = new ValidationDisplayWidget(this);
layout->addWidget(display);
// Update when validation changes
connect(validator, &EntityValidator::validationChanged,
display, &ValidationDisplayWidget::setValidationResult);
