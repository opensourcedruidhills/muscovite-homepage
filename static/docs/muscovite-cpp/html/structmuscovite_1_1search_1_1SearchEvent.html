Detailed Description

Transport-agnostic domain event for search indexing.

Contains the minimum information needed by the SearchIndexer to determine what action to take (upsert vs delete) and which DocumentMapper to invoke.

Generated NATS consumers convert muscovite::dba::EventContext to SearchEvent before calling SearchIndexer::process_event().
