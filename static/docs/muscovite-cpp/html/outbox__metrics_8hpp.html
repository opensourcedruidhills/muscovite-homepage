Detailed Description

Prometheus-style metrics for outbox publisher.

Provides simple in-memory counters and gauges for monitoring outbox publisher health. Follows the same pattern as grpc/interceptors/MetricsStore.

Metrics exposed:

  • muscovite_outbox_events_published_total{schema}
  • muscovite_outbox_events_failed_total{schema}
  • muscovite_outbox_events_retried_total{schema}
  • muscovite_outbox_processing_lag_seconds{schema}
  • muscovite_outbox_batch_size{schema}
  • muscovite_outbox_queue_depth{schema}
  • muscovite_outbox_broker_connected{schema}
