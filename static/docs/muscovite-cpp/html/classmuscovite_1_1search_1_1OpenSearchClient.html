Detailed Description

Low-level REST client for OpenSearch.

Provides typed methods for index management, document CRUD, search queries, and bulk operations. Uses libcurl internally (hidden behind PIMPL).

Thread safety: NOT thread-safe. Each thread should have its own client instance or use external synchronization. CURL handles are not shared across threads.

SearchConfig config;
config.host = "localhost";
config.port = 9200;
OpenSearchClient client(config);
// Create an index
auto idx_resp = client.create_index("patients", mapping_json);
// Index a document
auto doc_resp = client.index_document("patients", patient_id, patient_json);
// Search
auto result = client.search({"patients"}, query_json);
for (const auto& hit : result.hits) {
// process hit.source
}
