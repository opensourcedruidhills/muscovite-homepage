Detailed Description

NATS JetStream message broker implementation

Connects to NATS and publishes messages using JetStream for durability. Automatically reconnects on connection loss.

Thread Safety: This class is thread-safe. Multiple threads can call publish() concurrently. The underlying cnats library handles connection management.

Usage:

NatsConfig config;
config.url = "nats://localhost:4222";
NatsMessageBroker broker(config);
if (broker.connect()) {
auto result = broker.publish("muscovite.patient.PatientCreated", payload, event_id);
if (result == PublishResult::Success) {
// Event published to JetStream
}
}
