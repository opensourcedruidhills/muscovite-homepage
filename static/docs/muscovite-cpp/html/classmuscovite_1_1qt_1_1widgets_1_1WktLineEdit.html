Detailed Description

Single-line WKT input widget for simple geometries (POINT, short lines).

Features:

  • Real-time validation with visual feedback (green/red border)
  • Geometry type indicator icon
  • Optional "Pick on Map" button (for future integration)
  • Error tooltip on hover

Example: auto* wktEdit = new WktLineEdit(this); wktEdit->setAllowedType(GeometryType::Point); connect(wktEdit, &WktLineEdit::wktChanged, this, &MyDialog::onWktChanged);
