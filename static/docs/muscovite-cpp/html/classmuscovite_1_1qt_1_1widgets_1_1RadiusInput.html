Detailed Description

Distance input widget for radius-based spatial queries.

Features:

  • Single QDoubleSpinBox for distance in kilometers
  • Range: 0.001 km (1 meter) to 40,075 km (Earth circumference)
  • Clear labeling: "Radius" or custom label
  • Automatic validation

Layout: [ Radius | spinbox ] km
