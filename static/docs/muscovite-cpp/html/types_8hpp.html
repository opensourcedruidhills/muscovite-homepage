Detailed Description

Master include for all muscovite strong types.

This header-only library provides validated wrapper types for common domain-specific values. These types enforce validation at construction and provide type-safe alternatives to raw std::string.

Usage

#include <muscovite/types/types.hpp>
// Create from user input (validates)
auto email = muscovite::types::Email("user@example.com");
// Create from trusted source (skips validation)
auto email2 = muscovite::types::Email::from_trusted("db@example.com");
// Get underlying value
std::string s = email.value();

Validation

Each type has an is_valid() static method that can be used for pre-validation before construction. The regex patterns match those used in PostgreSQL DOMAIN CHECK constraints.

gRPC/Qt Integration

These types are designed to integrate with:

  • gRPC: Via to_string() for proto serialization
  • Qt: Via conversion to QString in muscovite::qt:: wrappers
  • PostgreSQL: CHECK constraints use identical regex patterns
