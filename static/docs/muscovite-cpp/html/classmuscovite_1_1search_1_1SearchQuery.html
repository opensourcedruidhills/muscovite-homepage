Detailed Description

Fluent builder for OpenSearch query DSL.

Constructs valid OpenSearch JSON query bodies using a type-safe interface. The builder accumulates clauses and produces the final JSON via to_json().

Query types:

  • full_text() — multi_match across all searchable fields
  • match() — single-field match query
  • term() — exact term filter (not analyzed)
  • range() — range filter on date/numeric fields
  • bool_query() — explicit bool with must/should/filter/must_not

Modifiers:

  • paginate() — from/size pagination
  • highlight() — field highlighting with tags
  • sort() — field sort order
  • aggregate() — terms aggregation (facets)
  • min_score() — minimum relevance threshold

Thread safety: NOT thread-safe. Build the query on a single thread, then pass the resulting JSON to OpenSearchClient.

See also
OpenSearchClient::search — consumes to_json() output
PagedSearchResult — wraps the response
