Detailed Description

template<typename Entity>
struct muscovite::types::TaggedId< Entity >

Compile-time-safe foreign key wrapper. TaggedId<Patient> is a distinct type from TaggedId<Order> — prevents cross-entity ID mixups that only fail at SQL FK violation time.
