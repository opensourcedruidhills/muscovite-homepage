Detailed Description

Authorization service for privilege checking.

This service handles group-based authorization:

  • Users belong to groups
  • Groups have privileges
  • Privileges define access to resources/actions

Privilege codes follow the pattern: {context}.{action} Examples: mineralogy.read, mineralogy.write, security.admin
