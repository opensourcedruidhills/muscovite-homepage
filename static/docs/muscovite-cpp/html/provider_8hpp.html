void muscovite::telemetry::init_telemetry ( const TelemetryConfig & config)

Initialize OTel TracerProvider AND MeterProvider.

When MUSCOVITE_TELEMETRY_AVAILABLE is defined, creates OTLP gRPC exporters for both traces and metrics to config.otlp_endpoint. When config.enabled == false, this is a complete no-op.
