Detailed Description

Thread-safe metrics storage for outbox publishers.

Each bounded context (schema) has its own set of metrics, allowing independent monitoring of different event streams.

Usage:

auto& metrics = OutboxMetrics::instance();
metrics.record_published("patient");
metrics.record_failed("patient");
metrics.record_lag("patient", 1.5); // 1.5 seconds lag
