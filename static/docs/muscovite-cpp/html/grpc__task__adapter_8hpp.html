Detailed Description

TaskTree-compatible adapter for async gRPC unary calls.

Bridges gRPC blocking stubs with Qt 6.11's QTaskTree framework. Provides GrpcCallTask (QObject with start()/done() contract) and GrpcUnaryTask (QCustomTask alias) for use in TaskTree recipes.

◆ GrpcUnaryTask

using muscovite::qt::GrpcUnaryTask = QtTaskTree::QCustomTask<GrpcCallTask>

TaskTree task type for gRPC unary calls.

This is a QCustomTask alias that uses QDefaultTaskAdapter<GrpcCallTask>. The adapter expects GrpcCallTask to have start() and done(DoneResult) — both are provided by the GrpcCallTask class.

Template parameters (from QCustomTask):
  • Task = GrpcCallTask
  • Adapter = QDefaultTaskAdapter<GrpcCallTask> (default)
  • Deleter = std::default_delete<GrpcCallTask> (default)
