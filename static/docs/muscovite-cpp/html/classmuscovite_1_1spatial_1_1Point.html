Detailed Description

Geographic point (latitude, longitude) Represents a single location on Earth using WGS84 coordinates.

Example: Point sf{37.7749, -122.4194}; // San Francisco auto dist = sf.distance_to(Point{40.7128, -74.0060}); // km to NYC
