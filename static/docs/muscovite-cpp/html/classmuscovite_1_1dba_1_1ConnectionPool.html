Detailed Description

Thread-safe PostgreSQL connection pool.

Usage:

auto pool = ConnectionPool::create(config).value();
{
auto conn = pool.acquire().value();
pqxx::work tx{conn.get()};
tx.exec("SELECT 1");
tx.commit();
} // Connection returns to pool
