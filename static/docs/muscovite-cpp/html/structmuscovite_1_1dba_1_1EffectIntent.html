Detailed Description

A typed intent to execute an external effect after transaction commit.

Produced by PLAN statements in DECIDE blocks. Persisted to the outbox table within the same transaction, then relayed asynchronously by OutboxPublisher via NATS.

The effect_type corresponds to an EFFECT declaration in the DSL. The payload is a JSON-serialized Proto message (typed per EFFECT).
