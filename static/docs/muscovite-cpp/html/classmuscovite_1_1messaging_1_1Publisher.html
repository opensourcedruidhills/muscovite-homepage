Detailed Description

Abstract publisher interface

Provides a transport-agnostic API for publishing messages. Generated code depends only on this interface; the concrete implementation (NATS, in-memory) is injected at construction time.

Thread Safety: Implementations MUST be thread-safe for concurrent publish().
