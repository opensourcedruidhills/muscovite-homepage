Detailed Description

Centralized theming singleton for Muscovite Qt applications.

Generated Qt Widgets code calls Theme::instance().primaryColor() etc. instead of embedding hardcoded color literals.

Usage:

// Optional: configure before first use
muscovite::qt::ThemeConfig cfg;
cfg.primaryColor = QColor("#FF5722");
muscovite::qt::Theme::configure(std::move(cfg));
// Access anywhere
auto color = muscovite::qt::Theme::instance().primaryColor();
widget->setStyleSheet(
QString("background-color: %1;").arg(color.name()));
