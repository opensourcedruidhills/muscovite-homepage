Detailed Description

Result of a synchronous validator invocation.

Invariant
When ok is true, message is empty. When ok is false, message contains a human-readable reason.

Typical usage in a sidecar implementation:

ValidationResult validate(const Patient& value) const override {
if (value.family_name().empty())
return {.ok = false, .message = "Family name must not be empty"};
return {.ok = true};
}
