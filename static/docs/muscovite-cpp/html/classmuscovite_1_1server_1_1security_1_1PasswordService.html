Detailed Description

Service for password hashing and verification using Argon2id.

This service implements secure password hashing following OWASP recommendations:

  • Algorithm: Argon2id (memory-hard, resistant to GPU/ASIC attacks)
  • Time cost: 3 iterations
  • Memory cost: 64 MB
  • Parallelism: 4 threads
  • Hash length: 32 bytes

Passwords are never stored in plaintext - only hashes are persisted.
