std::optional< ValidationError > muscovite::qt::grpc::extractValidationErrorFromContext ( const ::grpc::ClientContext & ctx)

Extract a ValidationError from a gRPC ClientContext's trailing metadata.

This is useful for synchronous gRPC calls where you have direct access to the ClientContext after the call completes. It parses the x-validation-* metadata headers to build a ValidationError.

Parameters

Returns
