Detailed Description

Authentication manager for built-in username/password authentication.

This class manages authentication state and provides signals for auth events. It communicates with the muscoviteserver security gRPC services.

Usage:

auto& auth = AuthManager::instance();
auth.configure(AuthConfig{.serverAddress = "localhost:50051"});
connect(&auth, &AuthManager::loginSucceeded, [](const Session& session) {
qDebug() << "Logged in as" << session.username;
});
connect(&auth, &AuthManager::loginFailed, [](const QString& error) {
qWarning() << "Login failed:" << error;
});
auth.login("admin", "password");
