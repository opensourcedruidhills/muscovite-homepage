Detailed Description

Qt-based infrastructure state machine for frontend circuit breakers.

Provides three states (Closed, Open, Half-Open) with Qt signals for UI integration. Generated FE circuit breakers derive from this class and override onOpen()/onHalfOpen()/onTrip() for plugin-specific behavior.

Signal Flow
  • circuitOpened() → disable plugin-dependent UI sections
  • circuitHalfOpened() → show "recovering..." indicator
  • circuitClosed() → re-enable plugin-dependent UI sections
