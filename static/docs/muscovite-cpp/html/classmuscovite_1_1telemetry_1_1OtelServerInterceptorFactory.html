Detailed Description

Factory for OpenTelemetry gRPC server interceptors

Creates interceptors that:

  • Start a span on POST_RECV_INITIAL_METADATA (method name as span name)
  • End span on PRE_SEND_STATUS, recording gRPC status code as attribute
  • Propagate trace context via gRPC metadata (W3C Trace Context)

When TelemetryConfig.enabled == false, creates a no-op interceptor that just calls Proceed(). When MUSCOVITE_TELEMETRY_AVAILABLE is not defined, also creates a no-op interceptor.
