Detailed Description

Compile-time bounded decimal template using StringLiteral NTTPs.

BoundedDecimal<Min, Max> enforces decimal range at construction time. Uses empty StringLiteral ("") as sentinel for half-open ranges. Stores value as double internally; parses bounds at construction time.
