Detailed Description

gRPC service infrastructure for muscovite

This library provides:

  • GrpcServer: Server lifecycle management
  • ServiceBase: CRTP base class for generated services
  • Type mapping: Convert between C++ and proto types
  • Error mapping: Map DbError/PoolError to gRPC status
  • Interceptors: Logging, metrics, error handling

Usage

Generated services inherit from ServiceBase and use the utilities:

#include <muscovite/grpc.hpp>
class MyServiceImpl : public muscovite::grpc::ServiceBase<MyServiceImpl>,
public myproto::MyService::Service {
public:
using ServiceBase::ServiceBase;
::grpc::Status GetById(...) override {
return execute_read([&](pqxx::transaction<>& tx) {
// Database operations
});
}
};
int main() {
// Create connection pool
auto pool = dba::ConnectionPool::create(config).value();
// Create and configure server
muscovite::grpc::GrpcServer server{grpc_config};
// Register service
MyServiceImpl service{pool};
server.register_service(&service);
// Start and wait
server.start();
server.wait();
}
