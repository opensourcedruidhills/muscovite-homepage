Detailed Description

Bulk reindex from QueryRepositories to OpenSearch.

The SearchReindexer performs a full reindex of all searchable entities:

  1. Drop and recreate each index with fresh mappings
  2. Paginated fetch from DocumentMappers (which call QueryRepositories)
  3. Build NDJSON bulk payloads
  4. Submit via OpenSearchClient::bulk()

This is used for:

  • Initial index population after deployment
  • Full reindex after mapping changes
  • Recovery from index corruption

Architecture:

SearchReindexer
│
├── DocumentMapper::fetch_page(offset, limit) → JSON documents
├── build_ndjson() → NDJSON string
└── OpenSearchClient::bulk() → BulkResponse
