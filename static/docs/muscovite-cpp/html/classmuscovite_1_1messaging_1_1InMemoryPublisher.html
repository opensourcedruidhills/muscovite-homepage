Detailed Description

In-memory publisher for unit testing

Thread-safe. Published messages are stored and queryable for assertions.

Usage:

InMemoryPublisher pub;
pub.connect();
pub.publish(envelope);
CHECK(pub.published().size() == 1);
CHECK(pub.published().front().subject == "muscovite.patient.PatientCreated");
