Detailed Description

Bulk reindexer for OpenSearch.

Iterates all registered BulkDocumentMappers, fetches pages of entities, and submits them to OpenSearch via the _bulk API. Supports progress callbacks for CLI reporting.

Usage:

OpenSearchClient client(config);
SearchReindexer reindexer(client);
auto mapper = std::make_shared<PatientBulkMapper>(query_repo);
reindexer.register_mapper(mapper);
auto result = reindexer.reindex_all(ReindexConfig{.batch_size = 500},
[](auto& type, auto processed, auto total) {
spdlog::info("Reindexing {}: {}/{}", type, processed, total);
});

auto muscovite::search::SearchReindexer::reindex_all ( const ReindexConfig & config = {},
const ReindexProgressCallback & on_progress = nullptr ) -> ReindexResult

Reindex all registered entity types.

For each mapper:

  1. Drop index (if recreate_indexes=true and index exists)
  2. Create index with mapper's mapping
  3. Paginated fetch → NDJSON bulk → submit
  4. Report progress via callback

Errors for one entity type do not prevent reindexing others.

Parameters
configReindex configuration (batch size, recreate flag)
on_progressOptional progress callback
Returns
Aggregate result with per-entity details
