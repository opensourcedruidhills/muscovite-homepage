Detailed Description

Coordinate input widget combining latitude and longitude spinboxes.

Features:

  • Two QDoubleSpinBox widgets (one per coordinate)
  • Built-in CoordinateValidator for range checking
  • Clear labeling: "Latitude" and "Longitude"
  • Automatic validation on value changes
  • Integration with validation framework

Layout: [ Latitude | spinbox ] [ Longitude | spinbox ]
