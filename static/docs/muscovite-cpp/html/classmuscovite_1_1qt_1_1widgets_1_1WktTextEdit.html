Detailed Description

Multi-line WKT input widget for complex geometries (POLYGON, MULTILINESTRING).

Features:

  • Multi-line text edit for large coordinate lists
  • Line numbers (optional)
  • Validation status bar at bottom
  • Format/prettify button
  • Character/coordinate count

Example: auto* wktEdit = new WktTextEdit(this); wktEdit->setAllowedType(GeometryType::Polygon); wktEdit->setWkt("POLYGON((0 0, 1 0, 1 1, 0 1, 0 0))");
