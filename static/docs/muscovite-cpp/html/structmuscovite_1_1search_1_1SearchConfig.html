Detailed Description

Connection configuration for an OpenSearch cluster.

Holds host, port, and optional basic authentication credentials. Typically constructed from environment variables or the generated server configuration.

Invariant
host is non-empty
port is in range [1, 65535]
See also
OpenSearchClient — uses this config for all REST operations
