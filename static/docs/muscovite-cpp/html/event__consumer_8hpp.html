Detailed Description

Event consumer interface for processing domain events from message brokers.

Provides a push-based event consumption model with handler routing, error handling, and consumer group support for horizontal scaling.

enum class muscovite::dba::HandlerResult
strong

Result returned by event handlers to control acknowledgment.

Enumerator
Ack 

Successfully processed, acknowledge the message.

Nak 

Temporary failure, retry later with backoff JetStream will redeliver after ack_wait expires

Term 

Permanent failure, do not retry Message is moved to dead-letter queue (if configured)

InProgress 

Long-running operation in progress, defer acknowledgment Handler is responsible for calling explicit ack/nak later
