Detailed Description

Thin validation result struct used by SyncVirtual validator plugins.

This is a Layer 1 framework type — no domain logic, no dependencies beyond the standard library. It is included by every generated validator base class (Layer 2) so that user-owned sidecar implementations can return a structured pass/fail result.
