Detailed Description

gRPC server lifecycle manager

Manages server startup, shutdown, and service registration. Uses PIMPL pattern for ABI stability.

Usage:

GrpcServer server{config};
server.register_service(&my_service);
server.start();
server.wait(); // Block until shutdown
