Detailed Description

spdlog sink that decorates log records with trace_id/span_id from the active OTel span and forwards to the OTel LogBridge.

When MUSCOVITE_TELEMETRY_AVAILABLE is not defined or config.enabled == false, this sink is a complete no-op.

Usage: register alongside the existing console sink in server_main.cpp
