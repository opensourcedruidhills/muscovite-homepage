Detailed Description

Validator for geographic coordinates.

Validates:

  • Latitude: [-90, 90]
  • Longitude: [-180, 180]
  • Decimal precision: 6 decimal places (±0.000001 degrees)

Can be used with QLineEdit or QDoubleSpinBox for immediate validation feedback.
