Detailed Description

Template for gRPC-based uniqueness checker.

This template simplifies creating entity-specific uniqueness checkers. Implementations provide the gRPC call logic.

Example implementation:

class PatientMrnUniquenessChecker : public GrpcUniquenessChecker {
public:
PatientMrnUniquenessChecker(std::shared_ptr<grpc::Channel> channel)
: stub_(PatientService::NewStub(channel)) {}
protected:
void doCheck(const QString& value, const QString& excludeId,
UniquenessCallback callback) override {
// Create and execute async gRPC call
CheckMrnUniquenessRequest request;
request.set_mrn(value.toStdString());
if (!excludeId.isEmpty()) {
request.set_exclude_patient_id(excludeId.toStdString());
}
// ... async gRPC execution with callback
}
private:
std::unique_ptr<PatientService::Stub> stub_;
};
